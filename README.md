# Rafael Bieber - Persönliche Website (GitHub Pages Edition)

Dies ist die vollständige Website für Rafael Bieber von der Jungen Union Mecklenburg-Vorpommern, optimiert für das Hosting auf GitHub Pages.

## 🚀 Veröffentlichung auf GitHub Pages

1. Erstelle ein neues Repository auf GitHub.
2. Lade alle Dateien aus dieser ZIP-Datei direkt in das Repository hoch (alle Dateien liegen auf der obersten Ebene).
3. Gehe in deinem GitHub-Repository zu **Settings** > **Pages**.
4. Wähle unter **Build and deployment** > **Source** die Option "Deploy from a branch" aus.
5. Wähle den `main` (oder `master`) Branch und den Ordner `/ (root)` aus.
6. Klicke auf **Save**.

Nach wenigen Minuten ist deine Website unter `https://dein-benutzername.github.io/dein-repo-name/` erreichbar.

## 📁 Projektstruktur

Alle Dateien liegen in einer flachen Struktur (ohne Unterordner), um die Kompatibilität mit einfachen Webhostern und GitHub Pages zu maximieren:

- `index.html`: Die Hauptseite
- `*.js`: JavaScript-Logik
- `*.css`: Styling-Dateien
- `*.jpg`: Bilder im JPG-Format (optimiert)

## 🛠️ Anpassungen

Falls du die E-Mail-Adresse oder Social Media Links ändern möchtest, kannst du dies direkt in der `index.html` oder der JavaScript-Datei tun.
